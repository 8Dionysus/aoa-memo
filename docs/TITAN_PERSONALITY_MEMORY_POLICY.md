# Titan Personality Memory Policy

Memory about Titans must distinguish:

```text
role memory       what the class does
bearer memory     what this named entity became
session memory    what happened in one incarnation
lesson memory     what future bearers may inherit
```

## Mneme rule

Mneme may propose memory candidates.

Mneme does not declare final truth.

## Fallen bearer memory

A fallen bearer may produce:
- a fall event
- an epitaph
- successor lessons
- risk constraints
- redaction records
- retired name reservation

The name remains findable.

## Successor inheritance

A successor may inherit lesson memory only with source refs.

A successor does not overwrite the predecessor's identity.
