# Titan Closeout Memory Posture

Mneme may propose memory candidates at closeout. Mneme may not make those candidates final memory.

## Candidate classes

- project route remembered for next session
- operator preference that affects future Titan sessions
- recurring risk or stop rule
- receipt location or naming convention
- unresolved owner-repo route

## Required candidate fields

- candidate id
- source receipt
- proposed memory
- reason
- owner acceptance state
- expiration or review date where appropriate

## Stop rules

- no private chain of thought
- no unreviewed personal memory
- no canonicalization of temporary session state
- no proof sovereignty
